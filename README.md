# V-iNCU

虚假的 iNCU 客户端，用以方便在桌面端（Windows/MacOS）调试 mincu 微应用。

## Platform

- MacOS, SwiftUI

- Windows, WinUI

- Browser, [Chrome Extension](https://developer.chrome.com/docs/extensions/mv3/) or [Tampermonkey](https://www.tampermonkey.net/documentation.php#_match)
